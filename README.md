# 🛒 E-Commerce Website

A simple **E-Commerce front-end website** built using **HTML, CSS, and JavaScript**.
The project demonstrates the basic structure and design of an online shopping platform where users can browse products and interact with a modern user interface.

---

## 📌 Features

* 🏠 Attractive homepage layout
* 🛍️ Product listing section
* 🧭 Easy navigation bar
* 📱 Responsive design for different screen sizes
* 🎨 Clean and simple UI design

---

## 🛠️ Tech Stack

* **HTML5** – Website structure
* **CSS3** – Styling and layout
* **JavaScript** – Basic interactivity

---

## 📂 Project Structure

```
E-commerce-website
│
├── index.html
├── style.css
├── script.js
└── assets / images
```

---

## 🚀 How to Run the Project

1. Clone the repository

```
git clone https://github.com/AmayaM27/E-commerce-website.git
```

2. Open the project folder

```
cd E-commerce-website
```

3. Run the project
   Simply open **index.html** in any web browser.
